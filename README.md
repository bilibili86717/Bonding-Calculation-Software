# 胶粘剂性能评估系统

一个基于网页的PWA应用，用于计算高铁车窗胶粘剂的性能参数。

## 核心功能

### 1. 参数输入
- 项目信息（名称、图号、胶粘结型号）
- 玻璃及安装参数（质量、面积、尺寸、角度、胶层厚度）
- 环境与载荷参数（风压、加速度）
- 胶粘剂性能参数（强度、热膨胀系数）
- 老化系数参数
- 安全系数要求（可自定义）
- 隧道及会车工况要求（可自定义）

### 2. 计算与结果展示
- 自动计算胶层应力、安全系数
- 显示详细计算过程（9个步骤）
- 工况评估结果

### 3. 报告导出

#### Word 报告（.docx 格式，真正的Word文档）
- 使用 docx.js 库生成标准 Microsoft Office 文档
- 包含：项目信息、输入参数截图、关键参数摘要、评估结果、详细计算过程、结论与建议、编审批
- 表格格式与模板完全一致
- 图片正确嵌入
- 支持在手机和电脑上用Word打开

#### PDF 报告
- 生成专业排版的HTML报告
- 自动弹出打印对话框，选择"另存为PDF"
- A4纸张格式
- 自动分页（重要章节在新页开始）
- 宋体字排版

## 技术栈

| 技术 | 用途 | 库/资源 |
|------|------|---------|
| PWA | 离线访问、安装为应用 | manifest.json + service-worker.js |
| Word导出 | 生成真正的.docx | docx.js v8 (CDN) |
| 文件下载 | 触发浏览器下载 | FileSaver.js (CDN) |
| 截图 | 捕获输入界面 | html2canvas (CDN) |
| PDF | 浏览器打印 | window.print() + 专业CSS |

## 文件结构

```
├── index.html          # 主页面（参数输入、计算、导出按钮）
├── app.js              # 计算逻辑、界面交互
├── export_report.js    # 报告导出（Word + PDF）
├── style.css           # 样式
├── manifest.json       # PWA清单
├── service-worker.js   # PWA离线缓存
└── icon.png            # PWA图标
```

## 使用方法

### 电脑端
1. 双击打开 `index.html`（推荐 Chrome 或 Edge 浏览器）
2. 输入所有参数
3. 点击 **"🖩 开始计算"**
4. 查看计算结果
5. 导出报告：
   - **导出Word报告** → 下载 .docx 文件，可用 Microsoft Word 或 WPS 打开
   - **导出PDF报告** → 打开新窗口，自动弹出打印对话框，选择"另存为PDF"

### 手机/平板端
1. 在浏览器中打开页面（可部署到Web服务器或直接打开本地文件）
2. 操作同电脑端
3. 报告可下载到手机或通过云盘同步

### 安装为PWA应用
- 在 Chrome/Edge 中，点击地址栏右侧的"安装"图标
- 可离线使用（首次需联网以加载CDN资源）

## 报告格式说明

报告严格按照模板格式生成，包含以下章节：

1. **标题** - 胶粘剂性能评估报告（居中）
2. **项目信息** - 项目名称、产品图号、胶粘结型号
3. **输入参数** - 输入界面截图（自动捕获）
4. **关键参数摘要** - 两列表格，7个参数
5. **评估结果** - 总体评估 + 3个安全系数
6. **隧道及会车工况安全系数** - 5个工况评估
7. **详细计算过程** - 9个编号步骤，每步含公式和计算
8. **结论与建议** - 评估结论文本
9. **编审批** - 姓名/日期、编制、审核、批准
10. **页脚** - 报告生成日期

## 注意事项

- Word报告使用了CDN加载的docx.js库，首次导出需联网
- PDF报告依赖浏览器的打印功能，不同浏览器输出可能略有差异
- 报告中所有数字格式与模板保持一致
